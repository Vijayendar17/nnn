# FocusFlow - Pomodoro Task Timer

FocusFlow is a lightweight single-page productivity app built with plain HTML, CSS, and JavaScript.

## Features
- 25-minute focus sessions
- 5-minute short breaks
- Start, pause, and reset controls
- Task list with add and complete toggle
- Daily completed-task counter

## Run
1. Open `index.html` in your browser.
2. Add tasks and start a focus session.

## Project Structure
- `index.html` - app structure
- `styles.css` - UI styling
- `app.js` - timer and task logic

## Notes
This project has no external dependencies and runs fully offline.
